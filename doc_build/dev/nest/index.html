#快速开始

#安装

#前提

Node.js（版本 >= 16）

#脚手架

npm i -g @nestjs/cli

#创建项目

使用  Nest CLI  构建的项目会创建一个初始的项目结构,并安装 node modules 和其他一些样板文件，然后创建一个  src/  目录，并增加几个核心文件。

nest new project-name

#启动项目

cd project
npm install
npm run start

在浏览器中打开  http://localhost:3000/
