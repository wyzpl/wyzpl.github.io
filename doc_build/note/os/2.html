#VSCode 编辑器设置

#插件

TIP

前端开发的一些常用插件，提高工作效率。

#基础

  • Chinese (Simplified) (简体中文)

#Git 增强

  • Git Graph：查看仓库的 Git Graph，并从图中轻松执行 Git 操作。
  • GitLens — Git supercharged：查看代码最近修改记录、修改人员
  • git-commit-plugin：自动生成git commit消息

#快速增效

  • Turbo Console Log：快速console.log()

  • Code Spell Checker：单词拼写检查

  • Color Highlight：高亮代码中颜色

  • Image preview：预览代码中图片

  • Path Intellisense：代码路径提示

  • SVG：预览 SVG

  • IntelliJ IDEA Keybindings：IDEA 风格快捷键

  • Code Translate：鼠标悬浮显示翻译

  • Bookmarks：在代码中进行标记，通过标记快速跳转

  • GitHub Repositories：通过仓库地址浏览代码

#风格统一

  • EditorConfig for VS Code：编辑器风格统一插件

    • 需在项目根目录配置.editorconfig

  • ESLint：代码风格检查工具

    • standard 规则

  • Prettier - Code formatter：代码格式化工具，可根据项目配置文件进行格式化风格统一

    • 文档
    • 支持的文件类型
    • 优先依据项目配置文件进行格式化

#主题

  • GitHub Theme：Github 风格主题；有暗色和亮色
  • vscode-icons：vscode 文件图标
  • Error Lens：提供行内报错提示

#其他

  • Draw.io Integration：画图插件
  • Partial Diff：代码对比
  • Live Server：快速开启服务器
  • i18n Ally：国际化翻译插件，可提高行内翻译提示
    • 文档
  • UnoCSS
    • 文档
    • 快速查阅属性

#Vue

  • Vue - Official

  • Vue 3 Snippets

  • Vue VSCode Snippets

#AI 增强代码提示

  • CodeGeeX：目前在用，免费功能够用

  • GitHub Copilot：Github 出品

  • Tabnine：代码提示过短（没有深度使用）

  • TONGYI Lingma：阿里出品

#编辑器设置

#在 VSCode 中开启嵌入式（内联）提示

在 vscode 设置中搜索inlayHint，开启对应需要提示的选项，即可开启内联提示

灰色部分为内联提示效果
