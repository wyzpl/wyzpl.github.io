## Nodejs

Nodejs 是一个基于 Chrome V8 引擎的 JavaScript 运行环境，使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。

Nodejs 的包管理器 [npm](https://www.npmjs.com/)，是全球最大的开源库生态系统。

### 安装方式

:::tip 建议
在工作中每个项目可能使用的是不同的 Nodejs 版本，或者想尝鲜 Nodejs 新出 API

所以建议通过版本管理工具安装 Nodejs
:::

**官网安装**

- LTS 版本: [https://nodejs.org/en](https://nodejs.org/en)

- 历史版本: [https://nodejs.org/en/download/](https://nodejs.org/en/download/)

**版本管理工具安装**

- [Volta](https://volta.sh/)「 不仅可以控制 Nodejs 的版本，也可以控制 yarn、npm、pnpm 等工具的版本 🌳 」

- [pnpm](https://www.pnpm.cn/)「 本质上是一个包管理器，也可以同时安装多个 Nodejs 版本 🌳 」

- [n](https://www.npmjs.com/package/n) 「 是一个 npm 包，只支持 macos 上使用 <Badge text="macos 选它" type="info" /> 」

- [nvm](https://github.com/nvm-sh/nvm) / [nvm-windows](https://github.com/coreybutler/nvm-windows)

### 包管理工具

- [npm](https://www.npmjs.com/)「 是全球最大的开源库生态系统，与 Nodejs 捆绑安装 」

- [pnpm](https://www.pnpm.cn/)「 比 npm 速度快，更节省磁盘空间 🌳 」

- [yarn](https://yarnpkg.com/)「 Facebook 开源的包管理器 」
